# Paprika

自用浏览器插件, 可以快捷搜索书签/标签页, 填充提示词到页面输入框。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd+I` (macOS) / `Ctrl+I` (Windows) | 打开命令面板 — 搜索标签页、书签、执行命令 |
| 在任意输入框输入 `/t` | 打开提示词选择面板 — 搜索并插入自定义提示词模板 |
| `Escape` | 关闭当前面板 |


## 提示词填充 (`/t`)

输入 `/t` 即可触发：

- 出现浮动搜索面板，展示已保存的提示词模板
- 输入关键词快速过滤（匹配名称、内容、标签）
- 选中后 `/t` 被替换为提示词内容，自动插入到光标位置
- 按 `Escape` 或点击外部关闭面板，`/t` 同时被移除

## 提示词管理

在popup弹窗中点击「提示词管理」进入独立管理页面，支持新建、编辑、删除提示词，以及按标签分类和全文搜索。
